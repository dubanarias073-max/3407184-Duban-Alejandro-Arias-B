🎯 Descripción

Este proyecto consiste en una ficha de información interactiva para un sistema de gestión de cementerios. La ficha permite mostrar información principal de los cementerios, propiedades de las tumbas, servicios ofrecidos y estadísticas relevantes. Incluye funcionalidades interactivas como cambio de tema, copiar información al portapapeles y mostrar/ocultar secciones.

📚 Conceptos ES2023 Aplicados

Durante la realización del proyecto, pude aplicar y reforzar los siguientes conceptos de JavaScript moderno:

✅ Variables con let/const: Definí constantes para datos que no cambian y variables para valores dinámicos.

✅ Template literals: Generé contenido HTML dinámico usando backticks y ${}.

✅ Arrow functions: Todas las funciones de eventos y cálculos se hicieron con arrow functions.

✅ Destructuring: Extraje información de objetos y arrays de manera clara y eficiente.

✅ Optional chaining (?.): Para acceder a propiedades sin errores si algún valor es null o undefined.

✅ Nullish coalescing (??): Para asignar valores predeterminados en caso de null o undefined.

🚀 Funcionalidades Implementadas

Información principal: Nombre del cementerio, ubicación, descripción y código identificador.

Propiedades específicas: Número de tumbas, tipo de servicios disponibles, estado del cementerio.

Lista de elementos relacionados: Servicios ofrecidos (ej: mantenimiento de tumbas, atención al visitante, ceremonias) con niveles o prioridad.

Estadísticas dinámicas: Número de visitas, porcentaje de ocupación de tumbas y rating del cementerio.
